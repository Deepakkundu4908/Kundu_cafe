## How to Test

1. **Start Server:**
   ```bash
   npm start
   ```
   Server runs on: `http://localhost:3002`

2. **Go to Admin Login:**
   ```
   http://localhost:3002/auth/admin/login
   ```

3. **Enter Credentials:**
   - Email: `admin@kunducafe.com`
   - Password: `Admin@123`

4. **Click "Login as Admin"**

5. **Access Admin Dashboard:**
   - You'll be redirected to `/admin`
   - Full admin panel is now available
